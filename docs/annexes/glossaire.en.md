# Glossary

The acronyms and terms you meet across AroundLink and international mobility.

| Term | Definition |
| --- | --- |
| **IRO** | *International Relations Office* — the team that manages mobility at an institution. |
| **DRI / Head of IR** | The strategic steering of international mobility. |
| **Erasmus+** | The European Union's mobility programme, which funds student exchanges among other things. |
| **EWP** | *Erasmus Without Paper* — the European network for digital exchange of mobility data between institutions. See [EWP network](../ewp/ewp.md). |
| **IIA** | *Inter-Institutional Agreement* — the Erasmus+ agreement signed between two universities. |
| **Bilateral agreement** | A direct cooperation agreement (outside Erasmus/EWP) negotiated between two institutions. |
| **Double degree** | A joint programme where the student earns a degree at both institutions. |
| **Paying mobility** | A *study abroad* mobility where the student pays fees to the host institution. |
| **OLA** | *Online Learning Agreement* — the digital learning agreement: the list of courses taken abroad and their recognition. |
| **Learning Agreement** | The learning agreement (see OLA). |
| **ToR** | *Transcript of Records* — the official transcript of the mobility. |
| **ECTS** | *European Credit Transfer System* — the European credit system (30 ECTS ≈ one semester). |
| **Factsheet** | An institution's practical fact sheet (deadlines, housing/visa contacts…), exchanged over EWP. |
| **Nomination** | The official proposal of a student to a host institution, which accepts or rejects it. |
| **Affectation / placement** | A student's confirmed placement on a destination, within a campaign. |
| **Wish** | A destination choice ranked by the student during a campaign. |
| **Campaign** | A framed application round (dates, levels, number of wishes) run by the IRO. |
| **Place distribution** | The breakdown of an agreement's places by year, level, period and field. |
| **CNR** | *Change Notification Receiver* — the EWP notification telling a partner that data changed. |
| **HEI** | *Higher Education Institution*; the *HEI ID* is its EWP identifier. |
| **SCHAC** | A standard institution identifier (based on its domain). |
| **Erasmus code** | An institution's official Erasmus code. |
| **ROR** | *Research Organization Registry* — a global open registry of research/education organisations, the basis of [Atlas](../atlas/atlas.md). |
| **ECHE** | *Erasmus Charter for Higher Education* — the charter that accredits an institution for the programme. |
| **EQF** | *European Qualifications Framework* — the European framework of qualification levels (e.g. EQF 6 = bachelor, 7 = master, 8 = doctorate). |
| **ISCED** | The international classification of fields of education (discipline). |
| **OUnit** | *Organizational Unit* — a faculty/department of an institution, in the EWP sense. |
| **Mobility grant** | The Erasmus+ financial support paid to the student. See [Mobility grants](../etablissement/bourse.md). |
| **Beneficiary Module** | The official Erasmus+ tool where an institution reports its mobilities. |
| **SSO** | *Single Sign-On* — signing in with an existing account (e.g. Microsoft/Azure). |
| **Multi-tenant (SaaS)** | The architecture where each client institution is isolated from others on a shared platform. |
